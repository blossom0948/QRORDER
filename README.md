# QRORDER

스마트 QR 테이블오더 MVP 웹사이트입니다.

## 화면 구조

- `index.html`: 손님용 주문 화면
- `admin.html`: 사장님용 주문/메뉴 관리 화면

손님 화면에는 관리자 기능이나 사이트 소개 영역을 노출하지 않고, 메뉴, 장바구니, 주문 확인, 내 주문 진행상황만 제공합니다.

## 기능

- 손님: 메뉴 조회, 검색, 장바구니, 함께 담기 좋은 메뉴 추천, 주문 접수
- 손님: 주문 완료 후 5초 카운트다운 뒤 주문 화면 자동 복귀
- 손님: 현재 주문한 메뉴의 접수, 조리중, 완료 진행상황 확인
- 손님: 진행 중인 주문 화면에서 물, 앞접시, 냅킨 직원 호출
- 사장님: 주문 대기, 조리중, 완료 상태 관리
- 사장님: 음식 주문과 직원 호출 요청 구분 확인
- 사장님: 메뉴 이미지 URL/파일 업로드, 가격, 품절 여부 설정
- 사장님: 메뉴 추가 및 삭제

현재 정적 MVP라 주문과 메뉴 설정은 같은 브라우저 origin의 `localStorage`에 저장됩니다. 실제 매장 배포에서는 Firebase Firestore 같은 실시간 DB를 붙이면 손님 기기와 사장님 기기 간 동기화가 안정적으로 동작합니다.
